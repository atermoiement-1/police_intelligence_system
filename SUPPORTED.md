# Supported Platforms

The following platforms, operating systems, and web browsers are supported for running the AIPAS application:

## Web Browsers
*   **Google Chrome**: Version 110 or higher.
*   **Mozilla Firefox**: Version 112 or higher.
*   **Apple Safari**: Version 16.5 or higher.
*   **Microsoft Edge**: Version 110 or higher.

## Operating Systems
*   **Windows**: Windows 10/11 (Local execution via Powershell/Uvicorn).
*   **Linux**: Ubuntu 20.04/22.04 LTS (Docker and native deployments).
*   **macOS**: macOS Monterey or newer.
