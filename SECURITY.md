# Security Policy

## Supported Versions
We support security fixes for the latest release:

| Version | Supported |
| :--- | :--- |
| v3.x | Yes (Active) |
| v2.x | No |

## Reporting a Vulnerability
Do not open GitHub issues for security-sensitive bugs or vulnerabilities. Instead, please report them directly to the development team at `security@aipas.gov.in`.

We will review your submission and respond within **48 hours** with an action plan for patching the vulnerability.
