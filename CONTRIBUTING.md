# Contributing to AIPAS v3.0

We welcome community contributions to help improve the AI Police Assistance System! Please read the guidelines below to ensure a smooth collaboration.

## 🚀 How to Contribute

### 1. Code Standards
* Use TypeScript for all frontend changes, adhering to the project's ESLint rules.
* Backend changes should be written in Python and conform to PEP 8 standards.
* Always write unit tests for new functionality using Vitest (frontend) or Pytest (backend).

### 2. Submission Process
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-feature-name`.
3. Commit your changes with clear, descriptive commit messages:
   ```bash
   git commit -m "feat(cases): add relationship graph navigation"
   ```
4. Push to your branch and submit a Pull Request (PR) targeting the `main` branch.

### 3. Pull Request Requirements
* Provide a clear description of the problem solved and changes made.
* Link to any relevant issues (e.g., `fixes #123`).
* Ensure all continuous integration (CI) tests pass successfully.
